Pra quem gosta de usar linha de comando, o Google lançou uma ferramenta ótima: <a link="http://code.google.com/p/googlecl/">GoogleCL</a>. Ele permite acessar diversos serviços pela linha de comando.<br /> <br /><a href="http://www.blogger.com/" title="Blogger">Blogger</a>, <a href="http://www.google.com/calendar" title="Google Calendar">Calendar</a>, <a href="http://googlesystem.blogspot.com/2008/09/google-contacts.html" title="Google Contacts">Contacts</a>, <a href="http://docs.google.com/" title="Google Docs">Docs</a>, <a href="http://picasaweb.google.com/" title="Picasa Web Albums: free photo sharing from Google">Picasa</a> e <a href="http://www.youtube.com/" title="YouTube">Youtube</a> podem ser usados através do GoogleCL, que por sua vez usa o <a href="http://code.google.com/p/gdata-python-client/">gdata-python-client</a>, que é um cliente pronto pra usar as APIs desses serviços.   <br /> <br />Já estou usando pra fazer esse post. Espero que dê certo.
